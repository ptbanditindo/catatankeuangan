# Panduan Menjalankan Schema Database di Supabase

## Langkah 1: Buka Dashboard Supabase
Buka link ini di browser:
```
https://supabase.com/dashboard/project/oikofuiiuabmxtswtldu
```

## Langkah 2: Buka SQL Editor
Di sidebar kiri, klik **SQL Editor** (ikon >_ )

## Langkah 3: Buat Query Baru
Klik tombol **New Query** (atau "+ New Query")

## Langkah 4: Copy Seluruh Isi schema.sql
Buka file `schema.sql` di project ini, copy seluruh isinya, paste ke SQL Editor.

## Langkah 5: Jalankan
Klik tombol **Run** atau tekan `Ctrl + Enter`

## Langkah 6: Verifikasi
Setelah berhasil, akan muncul:
- ✅ "Success. No rows returned" untuk setiap CREATE TABLE
- ✅ 7 tabel akan muncul di **Table Editor** (sidebar kiri → Table Editor)

## Selesai! 🎉
Setelah ini, refresh `http://localhost:3000` dan aplikasi siap digunakan.
